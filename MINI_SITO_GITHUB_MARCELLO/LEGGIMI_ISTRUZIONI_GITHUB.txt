
ISTRUZIONI per caricare il tuo MINI SITO su GitHub
--------------------------------------------------

1. Estrai il contenuto di questo ZIP in una cartella sul tuo computer:
   - index.html
   - cartella images/

2. Vai su:
   https://github.com/MarcelloMantinoMosaici/MarcelloMantinoMosaici.github.io

3. Clicca su "Add file" → "Upload files"

4. Trascina dentro TUTTO IL CONTENUTO della cartella:
   - Il file index.html
   - La cartella images/

5. Clicca in basso su "Commit changes"

✅ Dopo circa 1 minuto, il sito sarà visibile a:
https://marcellomantinomosaici.github.io

Buona fortuna artista!
